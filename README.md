# OpenBINN: Towards a Transparent Evaluation of Biologically Informed Neural Networks

----

[**Website**]() | [**arXiv Paper**]()

**OpenBINN** is an ongoing project dedicated to developing a comprehensive and lightweight library for systematically evaluating biologically informed neural networks (BINNs). Designed with a focus on transparency, reproducibility, and interpretability, OpenBINN aims to provide robust tools for evaluating BINN-specific explanation methods, fostering a deeper integration of biological insights into neural network research.

OpenBINN remains in active development, with the goal of becoming a community-driven platform that includes curated datasets, BINN models, and tailored evaluation metrics. Researchers and practitioners will soon be able to leverage an intuitive API to benchmark explanation methods and explore biologically relevant insights.

## Current Development Status
- OpenBINN is under active development. Early contributors are welcome to participate in shaping the platform.
- The first beta version is expected to include support for key BINN models, datasets, and interpretability metrics.

## Unique Features of OpenBINN (Planned)
- *Tailored to BINNs*: Provides tools and datasets specifically designed for biologically informed neural networks, ensuring compatibility with pathway-based and omics-driven models.
- *Comprehensive Evaluation Metrics*: Introduces interpretability metrics that align with biological principles, such as pathway coherence and gene importance concordance.
- *Community Collaboration*: Encourages open-source contributions to extend datasets, models, and metrics for a broader research impact.

## Roadmap for Development
1. **Core Library**: Establish the foundational framework for dataset handling, model integration, and evaluation workflows.
2. **Pre-trained BINN Models**: Include commonly used BINNs such as P-NET and pathway-based transformers for benchmarking.
3. **Evaluation Metrics**: Develop biologically meaningful metrics to assess model interpretability and explanation faithfulness.
4. **Documentation**: Provide extensive documentation and tutorials for users to easily integrate OpenBINN into their research.
5. **Beta Release**: Launch a beta version with a limited set of features for early feedback.
